## 文件说明

### index.json
包含所有64卦的索引信息:
- 卦号、卦名、别名
- 卦象符号(Unicode)
- 上下卦组成
- 对应的详细数据文件名

### trigrams.json
包含八卦的基础信息:
- 八卦符号、属性、五行
- 二进制表示(用于卦象计算)

### hexagrams/*.json
每个卦的详细数据文件,包含:
- **基本信息**: 卦号、卦名、拼音、别名、卦象符号
- **八卦组成**: 上卦、下卦、卦象符号
- **五行属性**: 整体五行、上下卦五行
- **卦辞**: 原文 + 白话文翻译
- **象辞**: 原文 + 白话文翻译
- **六爻爻辞**: 每一爻的原文 + 详细白话文解释
- **解读提示**: 运势、事业、感情、健康、建议(现代实用解读)
- **关键词**: 该卦的核心特质
